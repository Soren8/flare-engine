Flare (Free Libre Action Roleplaying Engine) is a simple game engine built to handle a very specific kind of game: single-player 2D action RPGs.
Flare is not a reimplementation of an existing game or engine. It is a tribute to and exploration of the action RPG genre.

Android version of Flare requires manual installation of engine data files.
Instructions on how to install default game called 'The Empyrean Campaign' can be found here:
https://github.com/flareteam/flare-engine/wiki/Android-Port
